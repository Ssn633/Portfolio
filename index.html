Web-based Piano

This interactive project delivers an intuitive interface where users can effortlessly play melodies. Weatherv1

Weatherv1 is a web-based application that allows users to easily check the current weather conditions in their location or any other location around the world. The application utilizes the BeautifulSoup API to web scrape weather information from google rather than using a weather API.

Languages used/Technologies used
Django HTML Web Scrapping Tailwind CSS Vercel
Live Demo Github repo
To-Do Manager

To-do is a web based task or project management system that allows users to create to-do lists and manage their tasks. Initially users have to create credentials to login to the system. After login, users can create their own to-do lists and add tasks to the lists. Users can also edit and delete the tasks.

Languages used/Technologies used
Django HTML PostgreSQL Tailwind CSS Vercel
Live Demo Github repo
ParaScope: Streamlined Paragraph Summarization And Topic Expansion Tool

ParaScope provides a solution that can solve mental exhaustion during the process of gathering information online. The system can provide a concise summary of any text/paragraph. This is achieved using an encoder that gathers a lot of contextual information from the input text and a decoder that chooses important parts of the text to include in the summary.

Languages used/Technologies used
Machine Learning Natural Language Processing Google Colab mT-5 Model
